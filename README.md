# 🎥 Video OCR Extractor

Система распознавания текста из видео. Извлекает субтитры, титры, газетные заголовки и другой текст из видеоряда.

## 🎯 Функционал

- Покадровая обработка видео с адаптивным интервалом (1 кадр в секунду)
- Предобработка изображений для улучшения распознавания:
  - Перевод в градации серого
  - Бинаризация
  - Инверсия
  - Масштабирование (x2) для мелкого текста
- Дедупликация текста (сравнение последовательностей через difflib)
- Добавление временных меток к каждому фрагменту (формат ММ:СС)
- Поддержка русского и английского языков

## 🛠️ Технологии

| Технология | Назначение |
|------------|------------|
| **Python 3.8+** | Основной язык |
| **OpenCV (cv2)** | Обработка видео и кадров |
| **Tesseract OCR** | Распознавание текста |
| **NumPy** | Работа с массивами изображений |
| **difflib** | Сравнение и дедупликация текста |
| **Regex** | Постобработка текста |
